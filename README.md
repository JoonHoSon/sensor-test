# BeagleBone Black, Raspberry Pi, Intel Edison별 센서 샘플 코드
## Raspberry Pi
### wiringPi 설치
[설치법](http://wiringpi.com/download-and-install/) 참조.

```
$ sudo apt-get update
$ sudo apt-get upgrade
$ sudo apt-get install git-core
$ git clone git://git.drogon.net/wiringPi
$ cd wiringPi
$ git pull origin
$ sudo ./build
```
